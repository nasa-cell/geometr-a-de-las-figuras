# geometr-a-de-las-figuras

Juego interactivo de geometría en navegador con modo estudio, batalla y selección de figuras.

## Características
- Menú principal y pantalla de estudio
- Mapa de planetas con figuras geométricas
- Batalla interactiva con preguntas y respuestas
- Música de fondo y efectos visuales

## Archivos principales
- index.html
- styles.css
- script.js
- gameData.js

## Ejecutar localmente
Abre el archivo index.html en tu navegador.
